Corpo do projeto

O objetivo é transformar o CSV local github_top_repositories.csv em uma narrativa visual sobre popularidade em software livre. A unidade de análise é o repositório; a amostra exibida prioriza os maiores valores de Stars Count e usa os domínios do dataset para contextualizar os projetos.

Visualizações possíveis

  • Ranking pesquisável dos repositórios mais populares.
  • Dispersão entre estrelas, forks, issues, tamanho e watchers.
  • Mapa de calor de correlação entre métricas numéricas.
  • Distribuição por domínio, linguagem, licença, ano de criação e tópicos.
  • Comparação entre domínios, linguagens e tipo de proprietário.
  • Análise de recursos ativados: wiki, pages, discussions, issues e projects.
  • Rede de coocorrência de tópicos para investigar comunidades tecnológicas.

Tecnologias sugeridas

  • D3.js para gráficos customizados, escalas, brush, tooltips e tabelas.
  • Vega-Lite para gráficos declarativos rápidos e reprodutíveis.
  • JavaScript ES Modules para organizar transformação e interação.
  • Node.js para pré-processar o CSV e gerar JSON enxuto.
  • HTML/CSS responsivo para apresentação sem backend.
  • Git/GitHub Pages para versionamento e publicação estática.
